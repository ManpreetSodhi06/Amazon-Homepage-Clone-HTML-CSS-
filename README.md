# 🛒 Amazon Homepage Clone (HTML & CSS)

This project is a static clone of the classic homepage of Amazon, built using pure HTML and CSS. It focuses on recreating the layout and design of the original Amazon interface without using JavaScript or frameworks.

---

## 🚀 Features

- 🧭 Classic Amazon-style navigation bar  
- 🔍 Search bar UI  
- 🏠 Hero banner section  
- 🛍️ Product grid layout  
- 🎨 Styled using pure CSS  
- ⚡ Lightweight and fast  

---

## 🛠️ Tech Stack

- HTML  
- CSS
- Git & GitHub
